# Tools Directory

Place external command-line tools in this directory. ARK-Retro-Forge does NOT download or bundle these tools - you must obtain them yourself from official sources.

## Required Tools

### chdman.exe (Recommended)
- **Purpose**: PS1/PS2/Dreamcast CHD compression/decompression
- **Minimum Version**: 0.261
- **Source**: https://www.mamedev.org/release.html
- **License**: BSD-3-Clause / GPL-2.0+

### Optional Tools

#### maxcso.exe
- **Purpose**: PSP/PS2 CSO compression
- **Source**: https://github.com/unknownbrackets/maxcso

#### dolphin-tool.exe
- **Purpose**: GameCube/Wii RVZ compression
- **Source**: https://dolphin-emu.org/

#### wit.exe (Wiimms ISO Tools)
- **Purpose**: Wii/WiiU image management
- **Source**: https://wit.wiimm.de/

## Security Notice

1. Download tools only from official sources
2. Verify checksums when available
3. Scan with antivirus software
4. Never run tools with elevated privileges unless absolutely necessary

## Verification

Run `ark-retro-forge doctor` to check which tools are found and their versions.
